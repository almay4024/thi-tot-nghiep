0 Lấy lệnh (Prompt) cho AI

Copy đoạn lệnh này, gửi cho AI kèm file đề gốc để lấy JSON.

Đóng vai trò là chuyên gia số hóa dữ liệu và giáo viên luyện thi Tốt nghiệp THPT. Tôi sẽ tải lên một tài liệu chứa đề thi. Nhiệm vụ của bạn là trích xuất và chuyển đổi thành tệp data.json. 1. Định danh và Tính điểm: - Tự động nhận diện môn học ghi vào "subject". BẮT BUỘC chỉ dùng các tên chuẩn sau để đồng bộ hệ thống: "Toán", "Vật lí", "Hóa học", "Sinh học", "Ngữ văn", "Lịch sử", "Địa lí", "Tin học", "Công nghệ", "GDKT & PL", "Tiếng Anh". - Ghi tên đề rõ ràng vào "title". - Điểm Phần 1 (Mặc định cho mọi môn): "p1_points": 0.25. - Điểm Phần 2 (Mặc định cho mọi môn): "p2_points": [0.1, 0.25, 0.5, 1.0]. - Điểm Phần 3 (Phụ thuộc môn học): Môn Toán đặt "p3_points": 0.5. Các môn khác đặt "p3_points": 0.25. 2. Xử lý Lời giải, Hình ảnh & BẢNG BIỂU (CỰC KỲ QUAN TRỌNG): - NẾU CÂU HỎI CÓ HÌNH ẢNH HOẶC BẢNG BIỂU: TUYỆT ĐỐI KHÔNG tự tạo bảng HTML. Phải coi bảng biểu là hình ảnh. Thêm "image": "[CẦN_CHÈN_ẢNH]" vào nội dung câu hỏi. - NẾU TÀI LIỆU CÓ LỜI GIẢI: Trích xuất vào "explanation". Nếu lời giải có đồ thị/BẢNG BIỂU, thêm "explanationImage": "[CẦN_CHÈN_ẢNH_GIẢI_THÍCH]". - NẾU KHÔNG CÓ LỜI GIẢI: Tự giải chính xác và viết lời giải chi tiết vào "explanation". 3. Xử lý Đáp án: - Trắc nghiệm nhiều đáp án: Dùng MẢNG (VD "ans": [0, 1]). - PHẦN 3 (Trả lời ngắn): ĐÁP ÁN BẮT BUỘC TỐI ĐA 4 KÝ TỰ (đã tính dấu "-" và ","). Ví dụ HỢP LỆ: "-1,3", "1693", "3,5". Số thập phân BẮT BUỘC dùng dấu phẩy. 4. QUY TẮC BẢO MẬT JSON VÀ TRÌNH BÀY TOÁN HỌC (LUẬT THÉP): - CẤM LỖI ESCAPE CHUỖI JSON: Mọi ký tự gạch chéo ngược ( \ ) trong công thức LaTeX BẮT BUỘC phải được nhân đôi ( \\ ). Ví dụ ĐÚNG: \\frac{1}{2}, \\sqrt{x}, \\Delta. Ví dụ SAI: \frac{1}{2}. - KHÔNG dùng dấu ngoặc kép ( " ) bên trong văn bản. Hãy dùng dấu ngoặc đơn ( ' ). - Công thức bọc trong $ (VD: $x^2 = 0$). Có khoảng trắng hai bên dấu quan hệ (Ví dụ: $x < y + 1$). - Phần 2: mảng "items" KHÔNG kèm chữ cái "a)", "b)". 5. Xử lý Câu hỏi chùm: SAO CHÉP toàn bộ đoạn thông tin/hình ảnh dùng chung và gắn trực tiếp vào đầu trường "q" của TỪNG câu liên quan. Cấu trúc JSON bắt buộc (Ví dụ): { "timeLimit": 3000, "subject": "Toán", "title": "Đề Khảo Sát", "p1_points": 0.25, "p2_points": [0.1, 0.25, 0.5, 1.0], "p3_points": 0.5, "part1": [ { "q": "Hỏi?", "image": "[CẦN_CHÈN_ẢNH]", "options": ["A", "B", "C", "D"], "ans": 0, "explanation": "Giải", "explanationImage": "[CẦN_CHÈN_ẢNH_GIẢI_THÍCH]" } ], "part2": [ { "q": "Hỏi?", "items": ["1", "2", "3", "4"], "ans": [true, false, true, false], "explanation": "Giải" } ], "part3": [ { "q": "Hỏi?", "ans": "-1,5", "explanation": "Giải" } ] }
