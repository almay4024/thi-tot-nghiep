0 Lấy lệnh (Prompt) cho AI

Copy đoạn lệnh này, gửi cho AI (Gemini/ChatGPT) kèm theo file đề gốc để lấy mã JSON.

Đóng vai trò là một chuyên gia số hóa dữ liệu và giáo viên luyện thi Tốt nghiệp THPT. Tôi sẽ tải lên một file tài liệu (PDF/Word/Ảnh) chứa đề thi mô phỏng định dạng năm 2026. Nhiệm vụ của bạn là đọc toàn bộ đề thi, trích xuất nguyên văn và chuyển đổi nó thành một tệp data.json. 1. Yêu cầu phân loại và tính điểm: Tự động nhận diện môn học và thiết lập trường "p3_points": - Đề Toán: Đặt "p1_points": 0.25 và "p3_points": 0.5. - Đề Thường (Lý, Hóa, Sinh...): Đặt "p1_points": 0.25 và "p3_points": 0.25. 2. Xử lý Hình ảnh (CỰC KỲ QUAN TRỌNG): - Nếu câu hỏi có chứa hình ảnh, đồ thị hoặc bảng biểu phức tạp, hãy thêm trường "image": "[CẦN_CHÈN_ẢNH]" vào đối tượng JSON của câu đó. - Tuyệt đối giữ nguyên văn chữ của đề bài, không được tự ý tóm tắt bỏ qua chữ. 3. Xử lý Đáp án: - Nếu file có đáp án: Dùng đáp án đó điền vào trường "ans". - Nếu không có đáp án: Tự giải, điền "ans" và ghi lời giải chi tiết vào "explanation". 4. Định dạng Phần 2 và Phần 3: - Phần 2 (Đúng/Sai): Trong mảng "items", KHÔNG ghi kèm các chữ cái "a)", "b)", "c)", "d)". - Phần 3 (Trả lời ngắn): Đáp án là một chuỗi (String). BẮT BUỘC dùng dấu phẩy (,) cho số thập phân (VD: "12,5" hoặc "-3,14"). Tuyệt đối không dùng dấu chấm. 5. Quy tắc Toán học: - Tất cả công thức toán/lý/hóa phải được bọc trong cặp dấu $ (ví dụ: $x^2 + 2x = 0$). 6. Tên đề thi: Viết ngắn gọn rõ ràng (VD: Đề khảo sát/Đề chính thức + Địa điểm + Môn thi + Mã đề + Ghi chú nếu có) Cấu trúc JSON bắt buộc (Ví dụ): { "timeLimit": 3000, "title": "[Tên đề thi]", "p1_points": 0.25, "p3_points": 0.25, "part1": [ { "q": "Nội dung câu trắc nghiệm?", "image": "[CẦN_CHÈN_ẢNH]", "options": ["A", "B", "C", "D"], "ans": 0, "explanation": "Giải thích" } ], "part2": [ { "q": "Nội dung câu đúng/sai?", "items": ["Phát biểu 1", "Phát biểu 2", "Phát biểu 3", "Phát biểu 4"], "ans": [true, false, true, false], "explanation": "Giải thích" } ], "part3": [ { "q": "Nội dung câu trả lời ngắn?", "ans": "-1,5", "explanation": "Giải thích" } ] }
