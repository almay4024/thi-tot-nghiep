0 Lấy lệnh (Prompt) cho AI

Copy đoạn lệnh này, gửi cho AI kèm theo file đề gốc để lấy mã JSON.

Đóng vai trò là chuyên gia số hóa dữ liệu và giáo viên luyện thi Tốt nghiệp THPT. Tôi sẽ tải lên một tài liệu chứa đề thi. Nhiệm vụ của bạn là trích xuất và chuyển đổi thành tệp data.json. 1. Định danh và Tính điểm: - Tự động nhận diện môn học ghi vào "subject". - Ghi tên đề rõ ràng vào "title" (VD: Đề thi thử Sở Hà Nội Lần 1 - Toán - Mã đề 101). - Môn Toán: "p1_points": 0.25, "p3_points": 0.5. Môn khác: "p1_points": 0.25, "p3_points": 0.25. - Cấu hình điểm Phần 2: "p2_points": [0.1, 0.25, 0.5, 1.0]. 2. Xử lý Lời giải & Hình ảnh (CỰC KỲ QUAN TRỌNG): - NẾU TÀI LIỆU GỐC CÓ LỜI GIẢI: Trích xuất NGUYÊN VĂN vào "explanation". Nếu lời giải có hình minh họa/đồ thị, thêm "explanationImage": "[CẦN_CHÈN_ẢNH_GIẢI_THÍCH]". - NẾU TÀI LIỆU GỐC KHÔNG CÓ LỜI GIẢI: Bạn tự giải chính xác và tự viết lời giải chi tiết vào "explanation". (Không tạo explanationImage). - NẾU CÂU HỎI CÓ HÌNH ẢNH (đồ thị, bảng biểu...): Thêm "image": "[CẦN_CHÈN_ẢNH]" vào nội dung câu hỏi đó. Mọi hình ảnh của các đáp án A, B, C, D đều coi như là ảnh của câu hỏi. 3. Xử lý Đáp án (Hỗ trợ Đa Đáp Án): - Trích xuất đáp án chuẩn xác. Nếu có nhiều đáp án được chấp thuận (do sai đề hoặc nhiều dạng hiển thị), hãy dùng MẢNG. - VD Trắc nghiệm: "ans": [0, 1]. Trả lời ngắn: "ans": ["1,5", "3/2"]. - Nếu chỉ có 1 đáp án: VD "ans": 0, hoặc "ans": "12,5". 4. Quy tắc Toán học và Trình bày: - Công thức toán/lý/hóa bọc trong cặp dấu $ (VD: $x^2 + 2x = 0$). - BẮT BUỘC có khoảng trắng hai bên các dấu quan hệ (như < , > , =, \\le , \\ge) để tránh lỗi biên dịch. Ví dụ: $x < y + 1$. - Phần 2 (Đúng/Sai): mảng "items" KHÔNG kèm chữ cái "a)", "b)". - Phần 3: Số thập phân BẮT BUỘC dùng dấu phẩy (VD: "12,5"). Cấu trúc JSON bắt buộc (Ví dụ): { "timeLimit": 3000, "subject": "Toán Học", "title": "Đề Khảo Sát Sở Hà Nội", "p1_points": 0.25, "p2_points": [0.1, 0.25, 0.5, 1.0], "p3_points": 0.5, "part1": [ { "q": "Nội dung câu trắc nghiệm?", "image": "[CẦN_CHÈN_ẢNH]", "options": ["A", "B", "C", "D"], "ans": [0, 1], "explanation": "Giải thích chi tiết", "explanationImage": "[CẦN_CHÈN_ẢNH_GIẢI_THÍCH]" } ], "part2": [ { "q": "Nội dung câu đúng/sai?", "items": ["Phát biểu 1", "Phát biểu 2", "Phát biểu 3", "Phát biểu 4"], "ans": [true, false, true, false], "explanation": "Giải thích chi tiết" } ], "part3": [ { "q": "Nội dung câu trả lời ngắn?", "ans": ["-1,5", "-3/2"], "explanation": "Giải thích chi tiết" } ] }

1 Nạp Dữ Liệu (Đề Thô hoặc Đề Đã Đóng Gói)

Kéo thả file .json vào đây

Hoặc click để chọn file từ máy tính

Dán trực tiếp Code JSON

Quản lý & Sửa nội dung đề

Chọn câu hỏi bị sai và sửa lại chữ/đáp án một cách độc lập.

2 Quản lý Hình Ảnh (Chèn mới / Thay thế)

Thao tác dán ảnh: Sử dụng công cụ chụp màn hình. Sau đó CLICK CHUỘT vào ô tương ứng bên dưới (viền sẽ sáng lên) và ấn Ctrl + V. Chú ý đọc kỹ nhãn màu tím để biết dán ảnh cho Câu Hỏi hay Lời Giải.
